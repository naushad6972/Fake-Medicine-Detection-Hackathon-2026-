# Fake-Medicine-Detection-Hackathon-2026
## Team Name
MedGuard AI

## Hackathon
PAN INDIA HACKATHON 2026

## Problem Statement
Fake medicines are a serious public health issue in India. Many people are unable to differentiate between genuine and counterfeit medicines, which can lead to severe health risks.

## Proposed Solution
This project is an AI-based Fake Medicine Detection System that analyzes medicine names and related details using NLP techniques and verifies them against a trusted dataset to identify whether a medicine is likely genuine or suspicious.

## Features
- Medicine name analysis
- Genuine vs Fake prediction
- Simple and user-friendly interface
- "More Research" option for additional verification

## Tech Stack
- Python
- NLP
- Machine Learning
- Streamlit (for UI)
- GitHub
